## Implementation Details
- Added container div with `mx-auto w-full max-w-md md:max-w-lg lg:max-w-xl` classes to all main pages
- Added same max-width constraints to bottom navigation component
- This creates responsive max-widths: 448px on mobile, 512px on medium screens, 512px on large screens
- Content is centered horizontally with auto margins
- Maintains mobile-first padding with `p-4 pb-24` for bottom navigation

## Impact
- Desktop users will now see a centered, mobile-style layout similar to mobile view
- Content width is constrained to be more readable on larger screens
- Bottom navigation also respects the same max-width constraints
- Consistent visual experience across all screen sizes
