## Perubahan

### Halaman Analisis - Saran Pola Makan
- **Tampilan Header Baru**: Mengubah header section "Saran Pola Makan" dengan gradient background hijau ke emerald yang menarik
- **Visualisasi Poin Saran**: Memecah teks saran yang panjang menjadi kartu-kartu visual yang mudah dibaca
- **Smart Parsing**: Sistem secara otomatis mendeteksi format konten:
  - **Heading** (Poin, Pola, Tips, dll): Ditampilkan dengan emoji icon dan background gradient hijau
  - **Numbered List** (1., 2., dll): Ditampilkan dengan badge angka berwarna biru-cyan

  - **Plain Text**: Ditampilkan dengan icon amplop kuning (💡)
- **Interactivity**: Setiap kartu memiliki efek hover shadow untuk feedback visual
- **Emoji Icons**: Menggunakan emoji makanan (🥗, 💪, 🍎, 🥦, dll) untuk heading agar lebih menarik
- **Dark Mode Support**: Tampilan tetap optimal di mode gelap dengan warna yang sesuai
