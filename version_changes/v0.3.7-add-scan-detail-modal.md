# Version 0.3.7 - Add Scan Detail Modal

## Overview
Menambahkan fitur untuk menampilkan detail informasi scan makanan ketika card diklik di halaman dashboard dan riwayat.

## Changes

### New Files
- `resources/js/components/scan-detail-modal.tsx`
  - Component modal untuk menampilkan detail informasi scan
  - Menampilkan gambar makanan dalam ukuran besar
  - Menampilkan nama makanan, deskripsi, dan confidence score
  - Menampilkan informasi waktu scan (tanggal dan jam)
  - Menampilkan nilai gizi lengkap (kalori, protein, karbohidrat, lemak)
  - Menampilkan informasi tambahan (serat, gula, natrium) jika tersedia
  - Menggunakan Dialog component dari shadcn/ui
  - Mendukung scrolling untuk konten yang panjang

### Modified Files

#### `resources/js/components/nutrition-card.tsx`
- Menambahkan prop `onClick` pada `NutritionCardProps`
- Menambahkan event handler `onClick` pada card dalam mode compact
- Menambahkan cursor pointer dan hover effect pada card yang bisa diklik
- Conditional class `cursor-pointer transition hover:border-primary/50 hover:bg-accent` hanya diterapkan jika onClick diberikan

#### `resources/js/pages/dashboard.tsx`
- Mengimpor `ScanDetailModal` component
- Menambahkan state `selectedScan` untuk menyimpan data scan yang dipilih
- Menambahkan state `isModalOpen` untuk mengontrol visibility modal
- Menambahkan fungsi `handleCardClick` untuk membuka modal dengan data scan yang dipilih
- Menambahkan prop `onClick` pada setiap `NutritionCard` di section "Scan Terbaru"
- Menambahkan `ScanDetailModal` component di bawah content utama dengan props:
  - `isOpen={isModalOpen}`
  - `onClose={() => setIsModalOpen(false)}`
  - `data={selectedScan}`

#### `resources/js/pages/history.tsx`
- Mengimpor `ScanDetailModal` component
- Menambahkan state `selectedScan` untuk menyimpan data scan yang dipilih
- Menambahkan state `isModalOpen` untuk mengontrol visibility modal
- Menambahkan fungsi `handleCardClick` untuk membuka modal dengan data scan yang dipilih
- Menambahkan cursor pointer dan hover effect pada gambar di setiap card riwayat
- Menambahkan event handler `onClick` pada gambar untuk membuka modal detail
- Menambahkan `ScanDetailModal` component di bawah content utama dengan props:
  - `isOpen={isModalOpen}`
  - `onClose={() => setIsModalOpen(false)}`
  - `data={selectedScan}`

## Features Added

1. **Scan Detail Modal**
   - Menampilkan informasi lengkap tentang scan makanan
   - UI yang bersih dan mudah dibaca
   - Mendukung informasi nutrisi lengkap
   - Responsive dan scrollable untuk konten yang panjang
   - Tombol close yang jelas di pojok kanan atas

2. **Clickable Cards**
   - Card di dashboard bisa diklik untuk melihat detail
   - Gambar di history bisa diklik untuk melihat detail
   - Visual feedback dengan cursor pointer dan hover effect
   - Non-intrusif (tetap mempertahankan fungsi edit/delete yang ada)

## Technical Details

### Component Structure
```
ScanDetailModal
├── Food Image (full width, 48px height)
├── Food Name & Confidence Score
├── Scan Info (date & time)
├── Macronutrients (grid 2x2)
│   ├── Kalori
│   ├── Protein
│   ├── Karbohidrat
│   └── Lemak
└── Additional Nutrition (if available)
    ├── Serat
    ├── Gula
    └── Natrium
```

### State Management
- `selectedScan`: Menyimpan data scan yang sedang ditampilkan di modal
- `isModalOpen`: Menyimpan state apakah modal sedang terbuka atau tertutup
- Modal close akan mereset kedua state ini

### User Experience
1. User mengklik card di dashboard atau gambar di history
2. Modal terbuka dengan animasi smooth
3. User dapat melihat semua detail nutrisi
4. User menutup modal dengan klik tombol X atau klik di luar modal
5. State di-reset dan modal tertutup
