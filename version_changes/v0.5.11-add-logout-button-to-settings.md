# Version 0.5.11 - Add Logout Button to Settings

## Release Date
2026-03-09

## Summary
Menambahkan tombol logout yang terlihat di halaman Pengaturan Profil. Sebelumnya, tombol logout hanya tersedia melalui menu dropdown di header (desktop) dan sidebar (mobile), namun aplikasi NutriiD menggunakan layout mobile yang hanya menampilkan BottomNav sehingga tombol logout tidak dapat diakses oleh pengguna.

## Problem

Aplikasi NutriiD menggunakan `AppMobileLayout` yang hanya menampilkan `BottomNav` untuk navigasi. Tombol logout sebelumnya hanya tersedia di:
1. **Desktop Header**: Dropdown menu dari avatar user di pojok kanan atas (melalui `app-header.tsx`)
2. **Mobile Sidebar**: Dropdown menu dari user info di sidebar (melalui `app-sidebar.tsx`)

Namun, kedua komponen tersebut tidak digunakan di `AppMobileLayout`, sehingga pengguna tidak memiliki cara untuk logout dari aplikasi.

Meskipun tombol logout sudah ada di komponen `user-menu-content.tsx` dan sudah diperbaiki di versi 0.5.9, tombol tersebut tidak terlihat di UI karena komponen yang menampilkannya tidak digunakan di layout mobile.

## Solution

Menambahkan tombol logout langsung di halaman Pengaturan Profil (`/settings/profile`) yang dapat diakses melalui BottomNav.

### File Changed

#### `resources/js/pages/settings/profile.tsx`

**Changes:**

1. **Add Imports:**
   ```tsx
   import { router } from '@inertiajs/react';
   import { LogOut } from 'lucide-react';
   import { Separator } from '@/components/ui/separator';
   import { logout } from '@/routes';
   ```

2. **Add Logout Section:**
   ```tsx
   <div className="space-y-6">
       <Separator />
       <div className="flex items-center justify-between">
           <div>
               <h3 className="text-lg font-medium">Keluar</h3>
               <p className="text-sm text-muted-foreground">
                   Keluar dari akun Anda
               </p>
           </div>
           <Button
               variant="outline"
               onClick={() => {
                   router.post(
                       logout().url,
                       {},
                       {
                           onFinish: () => {
                               router.flushAll();
                           },
                       },
                   );
               }}
               className="border-red-200 text-red-600 hover:bg-red-50 hover:text-red-700"
           >
               <LogOut className="mr-2 h-4 w-4" />
               Keluar
           </Button>
       </div>
   </div>
   ```

### Key Features:

1. **Prominent Location**: Tombol logout ditempatkan di bagian bawah halaman Pengaturan Profil, setelah tombol Hapus Akun
2. **Clear Visual Design**: Menggunakan warna merah (border-red-200, text-red-600) untuk menandakan bahwa ini adalah action destructive
3. **Icon**: Menampilkan icon `LogOut` dari Lucide React untuk visual clarity
4. **Hover Effect**: Memberikan feedback visual saat di-hover (bg-red-50, text-red-700)
5. **Proper Logout Logic**: Menggunakan `router.post()` untuk mengirim request POST ke `/logout` endpoint, kemudian `router.flushAll()` untuk membersihkan cache Inertia

### User Flow:

1. User membuka aplikasi NutriiD
2. User mengklik "Pengaturan" di BottomNav
3. User melihat halaman Pengaturan Profil
4. User scroll ke bagian bawah halaman
5. User melihat section "Keluar" dengan tombol "Keluar"
6. User mengklik tombol "Keluar"
7. Sistem mengirim POST request ke `/logout`
8. Laravel Fortify menghandle request dan menghapus user session
9. Cache Inertia dibersihkan
10. User di-redirect ke halaman login

## UI Design

### Tombol Logout:
- **Teks**: "Keluar"
- **Icon**: LogOut (dari Lucide React)
- **Warna Border**: Red-200
- **Warna Teks**: Red-600
- **Hover Background**: Red-50
- **Hover Text**: Red-700
- **Style**: Outline button

### Section "Keluar":
- **Judul**: "Keluar" (font-medium, text-lg)
- **Deskripsi**: "Keluar dari akun Anda" (text-sm, text-muted-foreground)
- **Layout**: Flexbox dengan justify-between (judul di kiri, tombol di kanan)
- **Separator**: Garis pemisah di atas section

## Impact

- ✅ Tombol logout sekarang terlihat dan dapat diakses di halaman Pengaturan Profil
- ✅ Menggunakan HTTP method yang sesuai (POST) ke endpoint `/logout`
- ✅ Membersihkan cache Inertia setelah logout
- ✅ UI/UX yang jelas dengan warna merah untuk action destructive

## Notes

- Tombol logout di `user-menu-content.tsx` tetap ada dan berfungsi, namun tidak terlihat karena komponen `app-header.tsx` dan `app-sidebar.tsx` tidak digunakan di `AppMobileLayout`
- Jika di masa depan aplikasi ingin mendukung desktop view, tombol logout di `user-menu-content.tsx` akan otomatis terlihat melalui header
- Penambahan tombol logout di halaman Pengaturan Profil adalah solusi yang paling tepat untuk mobile-first design karena:
  - User secara intuitif mencari pengaturan akun di menu Pengaturan
  - Tombol logout adalah bagian dari pengaturan akun
  - Lokasi yang jelas dan mudah diakses
