# v0.2.3 - Make Nutrition Cards Scrollable

## Date
2026-03-08

## Summary
Changed the layout of nutrition cards (Kalori, Protein, Karbo, Lemak) from a 2x2 grid to a horizontally scrollable layout for better mobile user experience.

## Problem Solved

**Issue**: The nutrition cards displayed in a 2x2 grid layout took up too much vertical space on mobile devices, making it harder to scan through food items quickly.

**Impact**: Users on mobile devices had to scroll more to view all nutrition information, and the layout was not optimized for smaller screens.

## Changes Made

### Components
- **Updated**: `nutrition-card.tsx`
  - Changed macronutrients section from `grid grid-cols-2 gap-3` to `flex gap-3 overflow-x-auto pb-2`
  - Added `min-w-[140px]` to each nutrition card to ensure they maintain minimum width
  - Added `flex-shrink-0` to icons to prevent them from being compressed
  - Added `scrollbar-hide` class for cleaner appearance

### Styles
- **Updated**: `app.css`
  - Added `.scrollbar-hide` utility class to hide scrollbars across browsers
  - Includes `-ms-overflow-style: none` for IE/Edge
  - Includes `scrollbar-width: none` for Firefox
  - Includes `::-webkit-scrollbar { display: none; }` for Chrome/Safari

### Technical Details

**Before:**
```tsx
<div className="grid grid-cols-2 gap-3">
    {macros.map((macro) => (
        <div key={macro.label} className="flex items-center gap-3 rounded-xl p-3 {macro.bg}">
            {/* content */}
        </div>
    ))}
</div>
```

**After:**
```tsx
<div className="flex gap-3 overflow-x-auto pb-2 scrollbar-hide">
    {macros.map((macro) => (
        <div key={macro.label} className="flex items-center gap-3 rounded-xl p-3 min-w-[140px] {macro.bg}">
            {/* content */}
        </div>
    ))}
</div>
```
