# v0.2.4 - Make Nutrition Info Modal Scrollable

## Date
2026-03-08

## Summary
Made the nutrition info modal content scrollable to ensure all text can be read when content exceeds the viewport height. Added custom scrollbar styling for better user experience. Also added a disclaimer about increased nutritional needs with higher physical activity levels.

## Problem Solved

**Issue**: Modal content for nutrition information cards (Kalori, Protein, Karbo, Lemak) was not scrollable, causing important information to be cut off when the content exceeded the available screen space.

**Root Cause**: The modal content container had no scroll functionality, and the modal itself could grow beyond the viewport height, making it impossible to access all information, especially on smaller screens.

**Impact**: Users could not read the complete nutrition information, particularly the "Kenapa manusia perlu?" section which contains detailed bullet points about why each nutrient is important.

## Changes Made

### Components
- **Updated**: `nutrition-info-modal.tsx`
  - Added `max-h-[85vh]` to `DialogContent` to limit modal height to 85% of viewport
  - Added `flex flex-col` to `DialogContent` for proper flexbox layout
  - Added `flex-shrink-0` to `DialogHeader` to keep header fixed
  - Added `overflow-y-auto pr-2 flex-1 custom-scrollbar` to content container
  - This allows content to scroll while keeping header visible
  - Added informational disclaimer box about increased nutritional needs with higher physical activity
  - Disclaimer styled with blue background and icon for visual emphasis

### Styles
- **Updated**: `app.css`
  - Added `.custom-scrollbar` utility class with custom scrollbar styling
  - Thin scrollbar (6px width) with rounded thumb
  - Light gray scrollbar for light mode, dark gray for dark mode
  - Hover effect on scrollbar thumb for better interactivity
  - Transparent track for cleaner appearance

### Technical Details

The implementation uses CSS flexbox to create a fixed header with scrollable content:

```tsx
<DialogContent className="max-w-lg max-h-[85vh] flex flex-col">
  <DialogHeader className="flex-shrink-0">
    {/* Header stays fixed */}
  </DialogHeader>
  <div className="space-y-4 mt-2 overflow-y-auto pr-2 flex-1 custom-scrollbar">
    {/* Content scrolls */}
  </div>
</DialogContent>
```

The custom scrollbar provides:
- `scrollbar-width: thin` for Firefox
- `::-webkit-scrollbar` styling for Chrome/Safari/Edge
- Dark mode support via `@media (prefers-color-scheme: dark)`

The activity disclaimer uses:
- Blue background (`bg-blue-50` / `dark:bg-blue-950/30`) for visibility
- Border styling (`border-blue-200` / `dark:border-blue-800`)
- Info icon from SVG for visual emphasis
- Responsive text colors for light and dark modes

## Notes

This enhancement improves user experience by ensuring all nutrition information is readable regardless of screen size. The custom scrollbar styling provides a polished, modern look that matches the overall design aesthetic while maintaining accessibility. The modal now adapts gracefully to different viewport sizes without cutting off important content.
