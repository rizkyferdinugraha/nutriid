# v0.5.10 - Fix PreventCaching Middleware Type Error

## Date
March 9, 2026

## Changes

### Fixed Type Error in PreventCaching Middleware
- **File**: `app/Http/Middleware/PreventCaching.php`
- **Issue**: Middleware returned `Illuminate\Http\JsonResponse` but return type only allowed `Response|BinaryFileResponse`
- **Error**: `TypeError: Return value must be of type Illuminate\Http\Response|Symfony\Component\HttpFoundation\BinaryFileResponse, Illuminate\Http\JsonResponse returned`
- **Solution**: Updated return type declaration to include `JsonResponse` and `RedirectResponse`

### Details
- Added import for `Illuminate\Http\JsonResponse`
- Added import for `Illuminate\Http\RedirectResponse`
- Updated return type from `Response|BinaryFileResponse` to `Response|JsonResponse|RedirectResponse|BinaryFileResponse`
- Updated PHPDoc to include all response types in Closure parameter

### Root Cause
The middleware was applied to Inertia.js routes, which return `JsonResponse` objects for XHR requests. The original return type declaration didn't account for this, causing a type mismatch error when Inertia returned JSON responses.
