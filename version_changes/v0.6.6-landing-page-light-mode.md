# Version 0.6.6 - Landing Page Light Mode

## Changes
- Removed all dark mode classes (`dark:`) from the landing page
- Landing page now explicitly uses light mode only
- Removed `dark:from-slate-950 dark:to-slate-900` from main container gradient
- Removed `dark:bg-slate-950/80` from navigation bar
- Removed `dark:bg-grid-slate-800/50` from hero section background
- Removed `dark:bg-slate-950/50` from "How It Works" section
- Removed `dark:bg-slate-950/80` from footer

## Rationale
Since the application only uses light mode (no dark mode CSS variables defined), removing the dark mode classes from the landing page ensures consistency and cleaner code.
