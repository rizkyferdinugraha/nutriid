# v0.3.2 - Redirect ke Halaman Utama Setelah Menyimpan

## Ringkasan
Menambahkan redirect otomatis ke halaman dashboard setelah data scan berhasil disimpan di halaman scan.

## Perubahan Frontend

### Pages
#### `resources/js/pages/scan.tsx`
- **Import `router` dari Inertia**
  - Menambahkan `router` ke import dari `@inertiajs/react`
  - Menggunakan `router.visit()` untuk navigasi

- **Fungsi `saveScan`**
  - Menambahkan `setTimeout` setelah notifikasi sukses
  - Redirect ke halaman dashboard (`/dashboard`) setelah delay 1.5 detik
  - Memberikan waktu kepada user untuk melihat notifikasi sebelum redirect

## Perilaku Baru

### Sebelumnya
1. User menekan tombol "Simpan"
2. Notifikasi "Berhasil Disimpan!" muncul
3. User tetap berada di halaman scan
4. User harus menekan tombol "Scan Makanan Lain" atau navigasi secara manual

### Sekarang
1. User menekan tombol "Simpan"
2. Notifikasi "Berhasil Disimpan!" muncul
3. User otomatis di-redirect ke halaman `/dashboard` setelah 1.5 detik
4. User dapat melihat data yang baru disimpan di dashboard

## Detail Teknis

### Implementasi Redirect
```typescript
// Setelah notifikasi sukses
setTimeout(() => {
    router.visit('/dashboard');
}, 1500);
```

### Alur Pengalaman Pengguna
1. **Analisis AI Selesai** → Muncul notifikasi "Analisis Selesai!"
2. **User atur persentase** → Slider untuk menentukan berapa yang dimakan
3. **User tekan Simpan** → Loading state "Menyimpan..."
4. **Data tersimpan** → Notifikasi "Berhasil Disimpan!" muncul
5. **1.5 detik kemudian** → Otomatis redirect ke `/dashboard`

## Keuntungan

### UX (User Experience)
- **Alur lebih natural**: Setelah menyimpan, user langsung diarahkan ke halaman utama
- **Feedback visual**: Notifikasi memberikan konfirmasi sebelum redirect
- **Waktu yang cukup**: 1.5 detik cukup untuk membaca notifikasi
- **Tidak perlu navigasi manual**: User tidak perlu menekan tombol tambahan

### Efisiensi
- Mengurangi langkah navigasi yang diperlukan
- User dapat langsung melihat hasil di dashboard
- Mempercepat workflow scanning makanan

## Catatan
- Redirect menggunakan Inertia.js `router.visit()` untuk navigasi SPA yang smooth
- Delay 1.5 detik memberikan waktu yang cukup untuk user membaca notifikasi
- Jika user ingin tetap di halaman scan untuk scan lainnya, mereka dapat menekan tombol "Scan Makanan Lain" sebelum redirect terjadi
