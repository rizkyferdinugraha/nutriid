  - AI: Chutes.ai API (Qwen/InternVL models)

### Design Decisions
- Color palette: Green primary (#22C55E), Orange secondary (#F97316), Cyan accent (#06B6D4)
- UI Style: Glassmorphism, mobile-first, bottom navigation
- Typography: Instrument Sans (existing)

### Database Schema Planned
- `food_scans` - Store scan results and nutrition data
- `daily_summaries` - Aggregate daily nutrition totals
- `nutrition_goals` - User targets (Phase 2)

### API Endpoints Planned
- `POST /api/food-scan` - Upload & analyze food image
- `GET /api/food-scan` - List user's scans
- `GET /api/food-scan/{id}` - Get scan detail
- `DELETE /api/food-scan/{id}` - Delete scan
- `GET /api/daily-summary` - Get today's summary

### Environment Variables Required
```env
CHUTES_API_URL=https://llm.chutes.ai/v1
CHUTES_API_KEY=your_api_key_here
CHUTES_MODEL=Qwen/Qwen2.5-VL-32B-Instruct
```
