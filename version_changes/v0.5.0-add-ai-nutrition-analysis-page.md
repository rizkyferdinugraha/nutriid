# v0.5.0 - Add AI Nutrition Analysis Page

## Overview
Menambahkan halaman analisis gizi dengan AI yang memberikan saran, informasi penting, dan rekomendasi kesehatan berdasarkan data nutrisi harian pengguna.

## What's New

### Backend
- **NutritionAnalysisService** (`app/Services/NutritionAnalysisService.php`)
  - Service baru untuk menganalisis nutrisi harian dengan AI
  - Mengambil data dari FoodScan dan NutritionProfile
  - Menghitung BMR (Basal Metabolic Rate) dan kebutuhan kalori harian
  - Menggunakan OpenRouter API untuk mendapatkan analisis AI
  - Output dalam bahasa Indonesia dengan 4 bagian:
    - Saran Pola Makan
    - Informasi Penting
    - Rekomendasi Kesehatan
    - Status Nutrisi

- **NutritionAnalysisController** (`app/Http/Controllers/NutritionAnalysisController.php`)
  - Controller baru untuk menangani analisis nutrisi
  - Endpoint `POST /api/nutrition-analysis/analyze` untuk menganalisis nutrisi
  - Endpoint `GET /analysis` untuk menampilkan halaman analisis

### Frontend
- **Analysis Page** (`resources/js/pages/analysis.tsx`)
  - Halaman baru untuk analisis gizi AI
  - Fitur:
    - Pemilihan tanggal analisis
    - Menampilkan ringkasan nutrisi harian
    - Menampilkan hasil analisis AI dengan 4 bagian
    - Status nutrisi dengan indikator warna (Baik/Cukup/Kurang)
    - Loading state dan error handling
    - Responsive design dengan glassmorphism style

### Navigation
- **Sidebar** (`resources/js/components/app-sidebar.tsx`)
  - Menambahkan menu "Analisis Gizi AI" dengan icon Brain
  
- **Bottom Navigation** (`resources/js/components/bottom-nav.tsx`)
  - Menambahkan menu "Analisis" dengan icon Brain
  - Mengatur ulang urutan menu untuk menampatkan analisis

### Routes
- **web.php** (`routes/web.php`)
  - Menambahkan route `GET /analysis` untuk halaman analisis
  - Menambahkan route `POST /api/nutrition-analysis/analyze` untuk API analisis

## Features

### AI Analysis Features
1. **Saran Pola Makan**: 2-3 paragraf analisis pola makan hari ini
2. **Informasi Penting**: Minimal 5 poin informasi yang perlu diketahui
3. **Rekomendasi Kesehatan**: Minimal 5 rekomendasi kesehatan
4. **Status Nutrisi**: Penilaian status dengan indikator visual

### Visual Design
- Status nutrisi dengan warna:
  - Hijau (Baik)
  - Kuning (Cukup)
  - Merah (Kurang)
- Icon yang sesuai untuk setiap status
- Card layout dengan shadow dan rounded corners
- Scrollable macro nutrient cards

## Technical Details

### BMR Calculation
Menggunakan persamaan Mifflin-St Jeor:
- Pria: `(10 * weight) + (6.25 * height) - (5 * age) + 5`
- Wanita: `(10 * weight) + (6.25 * height) - (5 * age) - 161`

### Daily Calorie Needs
- Menggunakan multiplier 1.2 untuk lifestyle sedentary
- `BMR * 1.2`

### AI Prompt
- Context includes: date, user profile, daily summary, food items
- Prompt dalam bahasa Indonesia
- Meminta output dengan format terstruktur
- Minimal 5 poin untuk informasi penting dan rekomendasi

## Usage
1. Navigate to `/analysis` or click "Analisis Gizi AI" in sidebar
2. Select date to analyze
3. Click "Analisis dengan AI" button
4. Review the analysis results:
   - Nutrition status card
   - Meal pattern advice
   - Important information
   - Health recommendations

## Notes
- AI analysis requires OpenRouter API key to be configured
- Analysis is personalized based on user's nutrition profile
- If no nutrition profile exists, AI provides general recommendations
- Daily summary is fetched first before AI analysis
