# v0.1.7 - Remove Dark Mode

## Ringkasan
Menghapus fitur Dark Mode dan menyisakan Light Mode saja.

## Perubahan yang Dilakukan

### 1. Hook Appearance
- **File**: `resources/js/hooks/use-appearance.tsx`
- **Perubahan**: Simplifikasi hook untuk hanya support light mode
  - Hapus dukungan `dark` dan `system` mode
  - `appearance` selalu mengembalikan `'light'`
  - `updateAppearance` menjadi no-op function
  - `initializeTheme` selalu menghapus class `dark` dan set color scheme ke `'light'`

### 2. Komponen & Halaman (Dihapus)
- **File yang dihapus**:
  - `resources/js/components/appearance-tabs.tsx` - Komponen switcher theme
  - `resources/js/pages/settings/appearance.tsx` - Halaman pengaturan tampilan
  - `resources/js/routes/appearance/index.ts` - Route untuk appearance

### 3. Routes
- **File**: `routes/settings.php`
- **Perubahan**: Hapus route `/settings/appearance`
- **Juga**: Hapus referensi ke `TwoFactorAuthenticationController` yang sudah dihapus di v0.1.6

### 4. Settings Layout
- **File**: `resources/js/layouts/settings/layout.tsx`
- **Perubahan**: Hapus menu "Tampilan" dari sidebar navigasi settings

### 5. Bootstrap App
- **File**: `bootstrap/app.php`
- **Perubahan**:
  - Hapus `HandleAppearance` middleware dari web middleware group
  - Hapus `appearance` dari encrypt cookies exception list

### 6. Middleware (Dihapus)
- **File**: `app/Http/Middleware/HandleAppearance.php` - Middleware untuk handle appearance

### 7. CSS Styles
- **File**: `resources/css/app.css`
- **Perubahan**:
  - Hapus `@custom-variant dark (&:is(.dark *));`
  - Hapus seluruh `.dark` class beserta CSS variables-nya
  - Hanya menyisakan light mode variables di `:root`

## Alasan
- Simplifikasi aplikasi dengan fokus pada satu mode tampilan
- Mengurangi kompleksitas codebase
- Light mode lebih sesuai dengan tema nutrisi/health yang cerah dan segar

## Dampak
- Tidak ada lagi pilihan theme switcher
- Aplikasi akan selalu menggunakan light mode
- Ukuran CSS berkurang karena tidak ada dark mode styles
- Performance sedikit meningkat karena tidak perlu theme switching logic

## Testing
- Pastikan aplikasi selalu menampilkan light mode
- Pastikan tidak ada error terkait appearance di console
- Pastikan navigasi settings berfungsi dengan baik
