# v0.1.8 - Migrate to OpenRouter

## Summary
Migrasi AI service dari Chutes.ai ke OpenRouter untuk akses model yang lebih beragam dan fleksibel.

## Changes

### AI Service Migration
- **Backend**: `app/Services/AIService.php`
  - Ubah API endpoint dari `https://llm.chutes.ai/v1` ke `https://openrouter.ai/api/v1`
  - Implementasi multiple model support dengan fallback
  - Update environment variable keys dari `CHUTES_*` ke `OPENROUTER_*`

### Environment Variables
- **Tambah**:
  - `OPENROUTER_API_URL` - URL OpenRouter API (default: `https://openrouter.ai/api/v1`)
  - `OPENROUTER_API_KEY` - API key untuk OpenRouter
  - `OPENROUTER_MODEL` - Model yang digunakan (default: `qwen/qwen3-vl-235b-a22b-thinking`)

- **Hapus**:
  - `CHUTES_API_URL`
  - `CHUTES_API_KEY`
  - `CHUTES_MODEL`

### Available Models
Implementasi 5 model OpenRouter dengan prioritas:
1. `qwen/qwen3-vl-235b-a22b-thinking` (primary/default)
2. `mistralai/mistral-small-3.1-24b-instruct:free` (free tier)
3. `qwen/qwen3-vl-30b-a3b-thinking`
4. `nvidia/nemotron-nano-12b-v2-vl:free` (free tier)
5. `bytedance-seed/seedream-4.5`

### Configuration Files
- **Update**: `config/services.php` - Tambah konfigurasi `openrouter`
- **Update**: `.env.example` - Update environment variables example

## Migration Steps

### 1. Update .env file
```env
# Hapus baris ini:
# CHUTES_API_URL=https://llm.chutes.ai/v1
# CHUTES_API_KEY=your_key
# CHUTES_MODEL=Qwen/Qwen2.5-VL-32B-Instruct

# Tambah baris ini:
OPENROUTER_API_URL=https://openrouter.ai/api/v1
OPENROUTER_API_KEY=your_openrouter_api_key
OPENROUTER_MODEL=qwen/qwen3-vl-235b-a22b-thinking
```

### 2. Clear Cache
```bash
php artisan config:clear
php artisan cache:clear
```

## Benefits
- Akses ke berbagai model AI berkualitas tinggi
- Pilihan model gratis (free tier) tersedia
- Fleksibilitas untuk switch model tanpa code changes
- Better integration dengan modern AI ecosystem

- Pastikan API key OpenRouter tersedia di file `.env`
- Default model menggunakan `qwen/qwen3-vl-235b-a22b-thinking` untuk hasil terbaik
- Model free tier tersedia untuk testing/development:
  - `mistralai/mistral-small-3.1-24b-instruct:free`
