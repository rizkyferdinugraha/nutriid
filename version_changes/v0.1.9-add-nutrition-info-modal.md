# v0.1.9 - Add Nutrition Info Modal

## Date
2026-03-08

## Summary
Added interactive nutrition cards in the daily summary section. Users can now click on any nutrition card (Kalori, Protein, Karbo, Lemak) to open a modal with detailed information about what that nutrient is, what the value represents, and why humans need it.

## Changes Made

### New Files
- `resources/js/components/nutrition-info-modal.tsx` - New modal component that displays detailed nutrition information with three sections:
  - **Apa itu [nutrient]?** - Explains what the nutrient is
  - **Nilai apa ini?** - Explains what the displayed value means
  - **Kenapa manusia perlu?** - Explains why humans need this nutrient with bullet points

### Modified Files
- `resources/js/components/daily-summary.tsx` - Updated to make nutrition cards clickable:
  - Added state management for selected nutrition and modal visibility
  - Added click handlers to open the modal when a card is clicked
  - Added keyboard support (Enter/Space) for accessibility
  - Added hover effect (scale animation) to indicate interactivity
  - Integrated the NutritionInfoModal component
