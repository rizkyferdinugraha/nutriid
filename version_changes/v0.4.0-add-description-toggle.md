# v0.4.0 - Tambah Tombol "Lihat Selengkapnya"

## Ringkasan
Menambahkan fitur expand/collapse pada deskripsi makanan dengan tombol "Lihat selengkapnya" dan "Lihat sedikit" agar pengguna dapat melihat deskripsi lengkap tanpa mengorbankan layout.

## Perubahan

### 1. `resources/js/components/nutrition-card.tsx`
- Import `useState`, `ChevronDown`, dan `ChevronUp` dari lucide-react
- Menambahkan state `isDescriptionExpanded` untuk mengontrol tampilan deskripsi
- Modifikasi elemen deskripsi:
  - Deskripsi terbatas 2 baris secara default (`line-clamp-2`)
  - Tombol "Lihat selengkapnya" dengan ikon ChevronDown saat collapsed
  - Tombol "Lihat sedikit" dengan ikon ChevronUp saat expanded
  - Tombol dapat diklik untuk toggle status expand/collapse

### 2. `resources/js/components/scan-detail-modal.tsx`
- Import `useState`, `ChevronDown`, dan `ChevronUp` dari lucide-react
- Menambahkan state `isDescriptionExpanded` untuk mengontrol tampilan deskripsi
- Modifikasi elemen deskripsi:
  - Deskripsi terbatas 3 baris secara default (`line-clamp-3`)
  - Tombol "Lihat selengkapnya" dengan ikon ChevronDown saat collapsed
  - Tombol "Lihat sedikit" dengan ikon ChevronUp saat expanded
  - Tombol dapat diklik untuk toggle status expand/collapse

## Alasan
Fitur ini memungkinkan pengguna melihat deskripsi lengkap jika diperlukan, sambil menjaga layout tetap rapi dan proporsional dengan ukuran gambar. Pengguna memiliki fleksibilitas untuk membaca deskripsi detail tanpa mengganggu tampilan keseluruhan.

## Fitur
- Deskripsi default terbatas 2 baris di nutrition-card
- Deskripsi default terbatas 3 baris di scan-detail-modal
- Tombol interaktif dengan ikon panah
- Smooth transition antara collapsed dan expanded state
- Tombol berwarna primary dengan hover effect

## File yang Dimodifikasi

- `resources/js/components/scan-detail-modal.tsx`
