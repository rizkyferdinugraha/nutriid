# v0.4.6 - Perbaiki Perhitungan Daily Summary dengan Consumed Percentage

## Ringkasan Perubahan
Memperbaiki perhitungan daily summary saat fitur koreksi nama makanan digunakan, agar memperhitungkan persentase makanan yang dikonsumsi.

## Masalah
Saat pengguna mengoreksi nama makanan melalui fitur correction, perhitungan daily summary tidak memperhitungkan `consumed_percentage`. Kode sebelumnya mengurangi dan menambahkan nilai nutrisi penuh, bukan nilai yang sudah dikalikan dengan persentase makanan yang dikonsumsi.

Contoh:
- Sebelumnya: Jika scan memiliki 500 kalori dan consumed_percentage 50%, saat dikoreksi daily summary dihitung seperti 500 kalori penuh (bukan 250)
- Seharusnya: 500 kalori × 50% = 250 kalori

## Solusi
Memperbaiki metode `correct()` di `FoodScanController` untuk:
1. Menghitung nilai nutrisi lama dengan memperhitungkan `consumed_percentage`
2. Menggunakan metode `getActualNutrition()` dari model FoodScan untuk menghitung nilai nutrisi baru
3. Update daily summary dengan selisih antara nilai lama dan baru yang sudah dikalikan dengan percentage

## File yang Diubah
- `app/Http/Controllers/FoodScanController.php`

## Detail Perubahan

### FoodScanController::correct()
**Sebelum:**
```php
$oldValues = [
    'calories' => $scan->calories,
    'protein' => $scan->protein,
    'carbohydrates' => $scan->carbohydrates,
    'fat' => $scan->fat,
];

// Update daily summary
$dailySummary->total_calories = $dailySummary->total_calories - $oldValues['calories'] + $scan->calories;
// ... dan seterusnya
```

**Sesudah:**
```php
$percentage = $scan->consumed_percentage / 100;
$oldValues = [
    'calories' => $scan->calories * $percentage,
    'protein' => $scan->protein * $percentage,
    'carbohydrates' => $scan->carbohydrates * $percentage,
    'fat' => $scan->fat * $percentage,
];

// Calculate new actual nutrition values (considering consumed_percentage)
$newValues = $scan->getActualNutrition();

// Update daily summary
$dailySummary->total_calories = $dailySummary->total_calories - $oldValues['calories'] + $newValues['calories'];
// ... dan seterusnya
```

## Dampak
- Perhitungan daily summary sekarang akurat saat fitur koreksi nama makanan digunakan
- Card ringkasan hari ini akan menampilkan data yang sesuai dengan persentase makanan yang dikonsumsi
- Progress bar di card nutrisi akan menampilkan persentase yang benar terhadap target harian
