# Version 0.5.1 - Fix CSRF Token and DOM Errors

## Perubahan

### Bug Fixes
1. **Perbaiki Error 419 CSRF Token**
   - Menambahkan header `X-CSRF-TOKEN` pada request ke API nutrition-analysis
   - Menambahkan header `X-Requested-With: XMLHttpRequest`
   - Mengambil CSRF token dari meta tag di HTML

2. **Perbaiki Error "Target container is not a DOM element"**
   - Menambahkan pengecekan safety di `app.tsx` untuk memastikan element root tersedia sebelum createRoot
   - Mengonfigurasi SSR entry point di `vite.config.ts` dengan menambahkan `ssr: 'resources/js/ssr.tsx'`

### File yang Dimodifikasi
- `resources/js/pages/analysis.tsx` - Tambah header CSRF token
- `resources/js/app.tsx` - Tambah pengecekan element root
- `vite.config.ts` - Tambah konfigurasi SSR entry point
